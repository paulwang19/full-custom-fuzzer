# 模組架構

1. 弱點預測器
  - input: 程式碼
  - output: 函式權重檔
2. 種子生成器
  - input: 使用說明文件、指定生成選項列表、原始碼 (optional)
  - output: 種子資料夾 (內含 N 個資料夾，每個資料夾含選項和種子檔案)
3. 模糊測試模組
  1. 插樁器
    - input: 目標原始碼、函式權重檔
    - output: 目標插樁執行檔、完整函式權重檔 (補齊呼叫路徑上的權重)、邊函式對應檔
  2. Fuzzer
    - input: 目標插樁執行檔、種子資料夾、邊函式對應檔
    - output: 佇列種子資料夾、崩潰結果資料夾
